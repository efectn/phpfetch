## Requirements

 - php ***(min 5.6)***
 
 Note: Only supports GNU/Linux.
 
 - Supporting DBMS: MySQL, PostgreSQL, MongoDB
 - Supporting Webservers: Apache, Nginx, Caddy, Lighttpd, Roadrunner

## Installation

    make install DESTDIR=/
